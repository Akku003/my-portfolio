PLACERA - Placement Management & Support System

Placera - An AI-powered placement management platform streamlining campus recruitment for 1,000+ students across three stakeholder roles.

Year

2026

Academic Project

Full Stack Web Application

Description

Placera unifies campus placement activities into a single AI-integrated ecosystem for Administrators, Students, and Alumni. I architected the multi-channel notification engine (Web Push, Gmail OAuth, Twilio WhatsApp), built an ATS resume checker supporting PDF/DOCX/OCR with weighted compatibility scoring, and developed a gamified DSA coding module backed by a self-hosted Judge0 instance with Gemini AI code review. Built with React, Node.js, Express, and PostgreSQL across 25 relational tables spanning eligibility automation, real-time analytics, and an AI-based placement readiness scoring system.
